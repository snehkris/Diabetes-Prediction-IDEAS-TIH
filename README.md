# Diabetes Prediction – Spring Internship 2026

## Project Overview
This project focuses on predicting diabetes using machine learning classification models.
The Pima Indian Diabetes Dataset is used for training and evaluation.

## Models Used
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Google Colab
